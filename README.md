## Usage

![](./screenshots/1.png)

### Supported patterns
You might add a custom pattern as your own pattern with the keywords (`title`, `url`, and shortened URL `surl`). The following patterns are supported by default:
```
title
[title](url)
title - url
url
surl
title - surl
[title](surl)
=HYPERLINK("url", "title")
```
